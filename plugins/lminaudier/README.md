# lminaudier Plugin

Personal Claude Code extensions.

## Available extensions

- **Skills** (`skills/`) — context injected into Claude automatically; invoked implicitly by Claude based on description
- **Commands** (`commands/`) — slash commands invocable as `/lminaudier:<name>`
- **Agents** (`agents/`) — subagent definitions
- **Hooks** (`hooks/`) — scripts that run on Claude Code hook events
